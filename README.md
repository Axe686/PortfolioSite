# Ink & Paper — редакционный сайт-визитка

Премиальный one-page сайт с пятью скролл-сценами направлений услуг. Стек: Next.js (App Router), TypeScript, Tailwind CSS, GSAP + ScrollTrigger, shadcn/ui.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Примечание

Репозиторий хранит только текстовые ассеты. При необходимости добавьте favicon вручную. 

## Сборка

```bash
npm run build
npm start
```
